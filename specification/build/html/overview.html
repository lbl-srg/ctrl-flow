2. OverviewÂ¶

The software relies on two main components.

  1. A graphical user interface for editing Modelica models in a diagrammatic form, see Section 3.2 and Section 3.3.

  2. A configuration widget supporting assisted modeling based on a simple HTML input form, see Section 3.4.

We envision a phased development where

  1. the configuration widget would be first implemented and integrated into an existing graphical editor for Modelica,

  2. the full-featured editor would be developed in a second phase, providing diagrammatic editing capabilities and integrating the configuration widget natively.
